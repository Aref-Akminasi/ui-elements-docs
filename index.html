Getting Started

  • 1. Create a new folder my-project
  • 2. Cd my-project
  • 3. Using the terminal run the following command
  • npm i ui-elements gulp gulp-purgecss gulp-sass sass â€“save-dev
  • 4. Create config.scss file in the root directory of my-project
  • 5. Create index.scss file in the root directory of my-project
  • 6. Import the config.scss file and the UI Elements library
  • @import 'config.scss';
    @import 'ui-elements/sass/index.scss';
  • 7. Create gulpfile.js file in the root directory of my-project
  • 8. Paste the following in gulpfile.js file in order to watch your changes and compile the styles.
  • const { src, dest, watch, series } = require('gulp');
    const sass = require('gulp-sass')(require('sass'));
    const purgecss = require('gulp-purgecss');

    function buildStyles() {
    return src('index.scss')
    .pipe(sass({ includePaths: ['./node_modules'] }))
    .pipe(purgecss({ content: ['*.html'] }))
    .pipe(dest('css'));
    }

    function watchTask() {
    watch(['config.scss', 'index.scss', '*.html'], buildStyles);
    }

    exports.default = series(buildStyles, watchTask);
  • 9. Create index.html file in the root directory of my-project
  • 10. link the index.html file to the created styles by Gulp (you don't have to create css folder with index.css, it will be auto generated).
  • <link rel="stylesheet" href="css/index.css" />
  • 11. Run npx gulp in the terminal, in order to watch your changes
  • npx gulp
  • 12. Add tags and UI Elements classes in the index.html

Configuration

In order to configure the library you can pre define the variables in the config.scss file

$primary: #222222; //default #007bff

In addition to colors, the other default variables

// spacing
$base-padding: 0.75rem;
$base-margin: 0.75rem;

// borders
$base-border-radius: 5px;
$base-border-thickness: 1px;

// box-shadow
$base-box-shadow: 0px 2px 10px rgba(0, 0, 0, 0.25);

// font sizes
$base-font-size: 1rem;
$font-size-sm: $base-font-size * 0.75;
$font-size-lg: $base-font-size * 1.5;
$font-size-xl: $base-font-size * 2;
$font-size-xxl: $base-font-size * 3;

// transition
$base-transition-duration: 0.3s;
